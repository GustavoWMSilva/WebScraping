# WebScraping

Projeto da disciplina de Coleta, Preparacao e Analise de Dados.

Conteudo principal:
- `tarefa1_wikipedia.py` e `tarefa1_wikipedia.ipynb`: Tarefa 1 com Wikipedia
- `tarefa2_single.ipynb`: Tarefa 2 com IMDb
- `tarefa2_imdb_base.ipynb`: base de estudo para a Tarefa 2
- `README_tarefa2_base.md`: guia de estudo da Tarefa 2
- `requirements_tarefa2.txt`: dependencias da Tarefa 2

## Como usar

Ative o ambiente virtual:

```powershell
.\.venv\Scripts\Activate.ps1
```

Instale as dependencias da Tarefa 2:

```powershell
pip install -r requirements_tarefa2.txt
```
